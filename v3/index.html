Customer Journey Analytics upgrade checklist

This questionnaire is designed to help your organization upgrade from an implementation centered around Adobe Analytics to an implementation focused on Customer Journey Analytics.

The checklist in its default state provides a standard upgrade method recommended by Adobe. It lets you follow a limited number of steps to immediately start getting value out of Customer Journey Analytics. Use the questionnaire below to tailor these upgrade steps to your implementation for a given property.
